Gst.Bin.prototype.add

function add(element:Gst.Element):Boolean {
    // Gjs wrapper for gst_bin_add()
}

Adds the given element to the bin. Sets the element's parent, and thus takes ownership of the element. An element can only be added to one bin.

If the element's pads are linked to other pads, the pads will be unlinked before the element is added to the bin.

<note> When you add an element to an already-running pipeline, you will have to take care to set the state of the newly-added element to the desired state (usually PLAYING or PAUSED, same you set the pipeline to originally) with Gst.set_state, or use Gst.sync_state_with_parent. The bin or pipeline will not take care of this for you. </note>

MT safe.

element

the Gst.Element to add

Returns

TRUE if the element could be added, FALSE if the bin does not want to accept the element.
