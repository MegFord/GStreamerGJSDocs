codec_utils_h264_get_level

function codec_utils_h264_get_level(sps:guint8, len:Number):String {
    // Gjs wrapper for gst_codec_utils_h264_get_level()
}

Converts the level indication (level_idc) in the stream's sequence parameter set into a string. The SPS is expected to have the same format as for codec_utils_h264_get_profile.

sps

Pointer to the sequence parameter set for the stream.

len

Length of the data available in sps.

Returns

The level as a const string, or null if there is an error.
