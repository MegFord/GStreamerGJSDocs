codec_utils_aac_caps_set_level_and_profile

function codec_utils_aac_caps_set_level_and_profile(caps:Gst.Caps, audio_config:guint8, len:Number):Boolean {
    // Gjs wrapper for gst_codec_utils_aac_caps_set_level_and_profile()
}

Sets the level and profile on caps if it can be determined from audio_config. See codec_utils_aac_get_level and codec_utils_aac_get_profile for more details on the parameters. caps must be audio/mpeg caps with an "mpegversion" field of either 2 or 4. If mpegversion is 4, the "base-profile" field is also set in caps.

caps

the Gst.Caps to which level and profile fields are to be added

audio_config

a pointer to the AudioSpecificConfig as specified in the Elementary Stream Descriptor (esds) in ISO/IEC 14496-1 (see below for a more details).

len

Length of audio_config in bytes

Returns

true if the level and profile could be set, false otherwise.
