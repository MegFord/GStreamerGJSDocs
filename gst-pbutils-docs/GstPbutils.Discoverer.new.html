GstPbutils.Discoverer.new

function new(timeout:Gst.ClockTime):GstPbutils.Discoverer {
    // Gjs wrapper for gst_discoverer_new()
}

Creates a new GstPbutils.Discoverer with the provided timeout.

timeout

timeout per file, in nanoseconds. Allowed are values between one second (#GST_SECOND) and one hour (3600 * #GST_SECOND)

Returns

The new GstPbutils.Discoverer. If an error occurred when creating the discoverer, @err will be set accordingly and null will be returned. If @err is set, the caller must free it when no longer needed using GLib.free.
