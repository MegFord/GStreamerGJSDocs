codec_utils_aac_get_profile

function codec_utils_aac_get_profile(audio_config:guint8, len:Number):String {
    // Gjs wrapper for gst_codec_utils_aac_get_profile()
}

Returns the profile of the given AAC stream as a string. The profile is determined using the AudioObjectType field which is in the first 5 bits of audio_config.

<note> HE-AAC support has not yet been implemented. </note>

audio_config

a pointer to the AudioSpecificConfig as specified in the Elementary Stream Descriptor (esds) in ISO/IEC 14496-1 (see codec_utils_aac_get_level for a more details).

len

Length of audio_config in bytes

Returns

The profile as a const string and null if the profile could not be determined.
