GstPbutils.Discoverer.prototype.discover_uri_async

function discover_uri_async(uri:String):Boolean {
    // Gjs wrapper for gst_discoverer_discover_uri_async()
}

Appends the given uri to the list of URIs to discoverer. The actual discovery of the uri will only take place if GstPbutils.start has been called.

A copy of uri will be made internally, so the caller can safely GLib.free afterwards.

uri

the URI to add.

Returns

true if the uri was successfully appended to the list of pending uris, else false
