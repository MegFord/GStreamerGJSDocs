GstAudio.AudioBaseSrc::create_ringbuffer

function vfunc_create_ringbuffer():GstAudio.AudioRingBuffer {
}

Create and return the GstAudio.AudioRingBuffer for @src. This function will call the ::create_ringbuffer vmethod and will set @src as the parent of the returned buffer (see Gst.set_parent).

Returns

The new ringbuffer of @src.
