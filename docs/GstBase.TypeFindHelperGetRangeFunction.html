GstBase.TypeFindHelperGetRangeFunction

This function will be called by type_find_helper_get_range when typefinding functions request to peek at the data of a stream at certain offsets. If this function returns GST_FLOW_OK, the result buffer will be stored in buffer. The contents of buffer is invalid for any other return value.

This function is supposed to behave exactly like a Gst.PadGetRangeFunction.
